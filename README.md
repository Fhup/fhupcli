自定义 vue 脚手架创建工具 fhupcli

安装
```
npm install fhupcli -g
```

创建vue项目
```
fhupcli create project_name
```

添加组件
```
fhupcli addcpn cpn_name [-d your_path]
```

添加页面
```
fhupcli addpage page_name [-d your_path]
```

添加路由
```
fhupcli addstore store_name [-d your_path]
```
